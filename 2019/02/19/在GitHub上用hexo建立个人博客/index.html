在GitHub上用hexo建立个人博客

Posted on 2019-02-19 |

所需工具

-Windows10 64位(若为32位,请自行下载安装部分文件)
-internet 连接(无需科学上网)

所需工具的准备

1.安装 Notepad++

推荐使用notepad++进行文本编辑,而非系统记事本

为方便起见,在此提供其64位安装包
Notepad++ for windows x64
无需安装在C盘

2.安装 Git

由于众所周知的原因,国内访问git网站较慢,在此,笔者提供百度云链接,请自行下载安装
Git for windows x64
下载完成后基本上直接点下一步即可,以下为个别安装提示图,供选择时参考


同样无需安装在C盘

3.安装 Node.js

因官网下载较慢,提供百度云链接
Node.js for windows x64
同样无需安装在C盘
注意,此处设置的用户名即你今后博客的地址一部分.记住用户名与邮箱

4.npm更换成淘宝镜像源

win10系统下,右键点击开始菜单,选择Windows PowerShell(管理员)
打开后输入

完成后检查,输入

若显示为之前输入的网址则成功

5.创建自己的GitHub帐号并新建库

自行浏览GitHub并创建即可
在个人页面选择新建库如下


然后按照下图填好


特别注意图中的yourusername填你的github用户名,不可更改
打开gitbash,输入
ssh-keygen -t rsa
然后敲三次回车
这样会在C:\Users\用户\.ssh文件夹中生成几个文件,打卡此路径(不同电脑可能不一样,刚刚三个回车之后有提示,仔细阅读即可)选择id_rsa.pub,右键选择用notepad++打开,复制其中的内容
然后回到浏览器中GitHub页面,选择右上角的个人菜单如下


选择设置,找到SSH选项,打开


选择新的SSH密钥
将之前复制的密钥粘贴进第二个框中即可


再点击添加
SSH key添加之后，就可以在本机git bash中进行测试，输入ssh -T git@github.com进行测试,若输出内容有successfully字样则成功

6.设置本地git的用户名与邮箱

win10系统下,右键点击开始菜单,选择Windows PowerShell(管理员),依次输入

其中星号处填入你的GitHub用户名和邮箱
最后,输入git config --list检查是否正确

安装Hexo

1.首先确定你博客本地的存放位置,例如:D:/blog目录下

2.在上面地址处文件夹空白处点击鼠标右键,选择Git Bash Here,在Git Bash窗口中输入npm install -g hexo,如下图:


(同样可以在开始菜单中选择Git Bash并打开,输入cd /d/blog(此处以D:/blog目录为例),回车后再输入npm install -g hexo回车即可)
然后输入npm install hexo --save回车,最后检查hexo是否安装成功,继续输入hexo -v,若输出类似于下图则成功

3.继续在Git Bash中输入hexo init,即hexo初始化成功

在站点目录找到_config.yml文件,右键选择用notepad++打开,在末尾找到

将其更改为

注意:其中的yourname应替换为你的GitHub用户名

4.检查hexo配置是否正确:

在Git Bash中输入hexo s,然后打开浏览器输入地址 http://localhost:4000,显示如下

(可选)5.使用NexT主题

打开NexT并下载压缩包
解压所下载的压缩包至站点(即刚刚确定的博客本地目录)的 themes 目录下,并将解压后的文件夹名称(hexo-theme-next-5.1.4)更改为 next
然后在站点目录找到_config.yml文件,右键选择用notepad++打开,找到theme: landscape改为theme: next

上传至GitHub

在站点目录打开Git Bash,依次输入hexo clean,hexo g,hexo d`即可
打开你的博客网址,你就可以看到属于你自己的博客了!
